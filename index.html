FAQ / Troubleshooting

How do I update my grenade dataset?

Go to: AppData/Roaming/Orbit/GrenadeHelper.txt

Append new spots manually from channels like #nadehelper-data

If corrupted, delete the file. Orbit will re-download the default dataset.

Or, place a GrenadeHelper.txt next to Orbit.exe (won't be deleted during updates).

Can't see the menu?

Let CS2 fully load before starting Orbit

Path: Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive\game\bin\win64\cs2.exe

Right-click → Properties → Uncheck "Disable fullscreen optimizations"

Do not use -Vulkan in launch options

Any fixes for "This app can't run on your PC"?

Go to: Windows Security → Virus & threat protection → Manage Settings → Add an Exclusion → Folder

Navigate to the Folder Path C:\Users\\AppData\Roaming\Orbit

Confirm the Selection Then: Restart your PC
